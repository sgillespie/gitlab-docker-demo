# GitLab Docker Demo
A demo showing GitLab running with
 * GitLab Server
 * GitLab Runners
   * Shell executor, for building docker images
   * Docker executor, for building web projects
 * Private Docker Registry

# Dependencies
Before running this, install
 * Docker
 * Docker Compose
